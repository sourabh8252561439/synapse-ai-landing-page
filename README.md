# Synapse AI – SaaS Landing Page

Modern AI SaaS landing page built with React, TypeScript, and Vite.

## 🚀 Live Demo
https://synapse-ai-saas-landing-page.vercel.app/

## 📌 Features
- Modern SaaS UI/UX Design
- Responsive Landing Page
- Pricing Plans Section
- Analytics Dashboard Preview
- Glassmorphism Design System
- High Conversion Layout

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- CSS3

## 💼 Purpose
This project demonstrates a production-level SaaS landing page focused on modern UI design and conversion optimization for AI-based products.

## 📱 Responsive
Fully optimized for mobile, tablet, and desktop devices.
